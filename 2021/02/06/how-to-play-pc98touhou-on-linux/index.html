在 Arch Linux 上运行东方旧作

曾几何时，灵梦还没有驯服野生的阴阳玉……

曾几何时，灵梦没有玄爷就不能飞……

曾几何时，灵梦还没有办法低速移动……

粗制滥造的游戏 logo，不明觉厉的剧情对白和随处可见的白点自机狙一起编织出了 PC98 上的原始幻想乡。

那么，在哪里才能玩到原汁原味的东方旧作呢？

一、下载东方旧作资源

我使用的是 KatyushaScarlet 氏发布的【Dosbox-X】东方旧作整合，读者可以自行下载旧作整合包。

二、删除无用的文件

我们使用的是 Linux，所以这个整合包中有几个文件我们并用不到。我们需要删除根目录下的「启动游戏.bat」、dosbox-x 文件夹中的 「dosbox-x.exe」文件和 「run.bat」 文件。

整合包内剩下的文件都是游戏相关的文件，其中，「font_cn.bmp」和「font_jp.bmp」两个文件是用于切换游戏的中/日文版本的。

三、下载安装 DOSBox-X

我使用的是 Arch Linux，所以在添加 ArchlinuxCN 源之后可以很方便地使用 pacman 包管理器安装 DOSBox-X。

如果你也使用 Arch Linux 并且已经添加了 ArchlinuxCN 源，那么你可以通过在终端输入 sudo pacman -S dosbox-x-git 来安装 PC98 模拟器。

其他发行版的小伙伴可以参考官网或者自行搜索源来进行安装。

四、编写 Shell 脚本

KatyushaScarlet 氏发布的整合包中包含了一个批处理文件，可以很方便地进入中/日版游戏游玩。但是我们的平台是 Linux，Bash 或者 Zsh 的语法和 Windows 中的批处理有一些区别，所以我们为了获得相同的体验，需要编写一个简单的 Shell 脚本来替代之前删除的「启动游戏.bat」。

五、为脚本添加可执行权限

在终端下输入 chmod 755 你编写的脚本文件名.sh 即可。

六、开始游玩

对于大部分读者来说，经过上述步骤，现在应该可以直接通过执行脚本文件来游玩东方旧作了。但是对于机器比较特殊的人来说，可能还需要更改一下上述脚本，来获得更好的 HiDPI 体验。

在 Windows 上的 DOSBox-X 似乎可以自适应缩放游戏画面，但是在 Linux 上却不行，所以我们需要根据自己的屏幕来设置相应的画面缩放。相比于 DOSBox，DOSBox-X 还包含了一个下拉菜单用于实时更改设置。不过我们玩的游戏比较多，如果每次打开都要在下拉菜单那里点来点去非常麻烦，所以需要将缩放参数写进 conf 文件内。
